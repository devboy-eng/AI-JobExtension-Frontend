üêõ Common Issues & Fixes

Issue: Button doesn't respond

Causes:

  • Extension not properly loaded
  • LinkedIn page still loading
  • Content script errors

Fixes:

  1. Open Chrome DevTools (F12) ‚Üí Console
  2. Look for JavaScript errors in red
  3. Reload extension in chrome://extensions/
  4. Refresh LinkedIn page

Issue: "Job Title Not Found"

Causes:

  • LinkedIn changed HTML structure
  • Page not fully loaded
  • Wrong LinkedIn page type

Fixes:

  1. Wait 5-10 seconds for page to load
  2. Scroll down to ensure all content loads
  3. Try a different LinkedIn job posting
  4. Run debug script to see what data is available

Issue: Backend connection failed

Fixes:

  1. Check terminal - backend should be running
  2. Test health endpoint: http://localhost:4003/health
  3. Restart backend: PORT=4003 node start-backend.js
  4. Check firewall/antivirus blocking localhost
